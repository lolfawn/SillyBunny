# Agentic Experiment Template

Use this for each `staging` experiment.

## Title

Short name for the experiment.

## Goal

What capability are we trying to add or improve?

## Inspiration

What part of Aventuras or other agentic systems are we drawing from?

## Files touched

- List the main files expected to change.

## Hypothesis

What should improve if this works?

## User-visible change

What should a user notice in the UI or generation behavior?

## Prompt / orchestration impact

- Does it change pre-generation behavior?
- Does it change post-generation behavior?
- Does it add a new service or tool?

## Data impact

- Does it change chat metadata?
- Does it change lorebook format?
- Does it need migration?

## Validation

- Manual test steps
- Example chat scenario
- Failure case to watch for

## Rollback plan

How do we disable or revert this if the behavior is noisy, wrong, or too costly?
